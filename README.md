# __Перетворювач графічної таблиці у формат excel-таблиці__
## Встановлення
1. Скачати репозиторій
2. Розархівувати у бажану папку
3. Відкрити PyCharm/будь-яке інше IDE та активувати віртуальне середовище в терміналі за командою __.\\.venv\Scripts\activate__
4. Встановити необхідні залежності в терміналі за командою __pip install -r requirements.txt__
5. Перемістити скріншот у папку з __main.py__
6. Запустити __main.py__ та у діалоговому вікні вказати повну назву скріншоту з його форматом (schedule.jpg наприклад)

## Принцип роботи

&emsp;&emsp;Код виконує перетворення графіку, що відображенний на скріншоті у вигляді графічної інформації в зручну таблицю xlsx формату.
На прикладі вище маємо фото, яке нам необхідно представити у форматі, що буде зручним для сприйняття комп'ютером.
Щоб мати змогу проаналізувати зображені дані використаємо __Computer Vision (OpenCV)__ та задамо алгоритм, за яким код читатиме оригінальну таблицю.
<br>
<br>
![debug_grid](https://github.com/user-attachments/assets/76fdff97-a9f2-4872-baac-e87a5d2a55ca)
<br>
<br>
&emsp;&emsp;Згідно заданого алгоритму поверх оригінальної таблиці створюється додаткова маска - сітка прилипання, що фіксується за кожною лінією оригінальної таблиці.
Вона необхідна для чіткого визначення межі кожної з комірок, компенсуючи нерівномірність таблиці для подальшого аналізу кольору кожної з комірок.
Наступним кроком по створенній поверх сітці прилипання проводиться аналіз визначених комірок на наявність кольорових маркерів відключення завдяки фільтру насиченості (сатурації), що дає змогу ігнорувати як світлі клітини, так і повністю чорні, що необхідно для зменшення похибки вихідного результату.
